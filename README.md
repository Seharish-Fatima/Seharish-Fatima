## about

I like building the kind of projects that start with

> *"I wonder if I could..."*

and somehow end with thousands of lines of code.

One week I'm training machine learning models.

The next I'm compiling C++ to WebAssembly.

Then I disappear into a rabbit hole and come back with quantum chess because apparently regular chess wasn't chaotic enough.

I'm fascinated by algorithms, AI, systems programming, data visualization, and interactive products that make people want to click *just one more button.*

I don't care much about collecting dozens of unfinished repositories.

I'd rather ship something polished, learn a ridiculous amount building it, and then immediately chase the next interesting problem.
